Manual Angular

Introdução sobre Angular

O Angular é um poderoso framework mantido pelo Google para o desenvolvimento de aplicações web dinâmicas e interativas. Ele utiliza TypeScript para construir aplicações do lado do cliente e oferece uma abordagem de programação orientada a componentes.

Versões do Angular

Atualmente, o Angular possui várias versões, sendo a mais recente a Angular 13. É importante estar ciente das versões disponíveis para aproveitar os recursos mais recentes e manter a compatibilidade com o ecossistema Angular.

Instalando o Angular CLI

O Angular CLI (Command Line Interface) é uma ferramenta que facilita a criação e gerenciamento de projetos Angular. Para instalá-lo, utilize o comando:

npm install -g @angular/cli

Verificando a Instalação

Após a instalação, você pode verificar se o Angular CLI está corretamente configurado com o comando:

ng --version

Criando um Projeto Angular

Para criar um novo projeto Angular, utilize o comando:

ng new nome-do-projeto

Isso criará uma nova pasta com a estrutura inicial do projeto Angular.

Executando Aplicações em Angular

Para iniciar um servidor local e visualizar sua aplicação Angular, utilize o comando:

ng serve

Isso iniciará o servidor de desenvolvimento e você poderá acessar a aplicação no seu navegador.
